NameGame

A simple game that tests how well versed you are with names.
Correct Number is: 0

The current letter is:






Instructions: The game tests to see how many first names you can remember in a limited time environment. Initially, the system will choose a random letter to start off the game. You need to type in a given name that starts with the letter (and click enter after to submit). The last letter of the name that you type will become the new letter for which you need to enter a name. You have 10 seconds for each name. Repetitions are not allowed. Good Luck!
